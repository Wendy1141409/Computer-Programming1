使用二維陣列來解決以下問題。某家公司有四位銷售人員(編號1到4)，他們銷售五種不同產品(編號1到5)。每天一次，每位銷售員繳交便條紙，寫出當天售出不同產品的數量，每種產品一張便條紙。每張便條紙包含以下資料:
	a) 銷售人員編號
	b) 產品編號
	c) 該產品總銷售金額
因此每位銷售員每天必須繳交0到5張關於各種產品銷售金額的便條紙。假設上個月的所有銷售便條紙已經收集完成。請撰寫一個程式，能夠輸入每張便條紙的銷售資訊，並按照每位銷售員和產品來統計銷售金額的總和。所有銷售金額必需存放在一個二維陣列中。處理完所有資料後，請將結果用表格形式顯示出來，每一列代表某個特定銷售員，而每一行代表某個特定產品。將每一行的值相加，可以得到該產品上個月的總銷售金額，將每一列的值相加，可以得到該銷售員上個月的銷售總額。在表格右方顯示銷售員的總金額，在表格下方顯示該產品的總金額。
